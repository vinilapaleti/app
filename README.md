# 🌤️ Weather in a Click

A beautiful real-time Weather Dashboard built with **Streamlit** and Open-Meteo API.

### ✨ Features
- Search weather for any city in the world
- Current conditions with "Feels like" temperature
- Interactive hourly temperature chart (next 24 hours)
- Clean 7-day forecast table
- Animated moving clouds background
- Fully responsive (works great on mobile)
